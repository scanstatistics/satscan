
Building with Visual C++ 2010 Notes
-----------------------------------

A symbolic link for both Boost and Java JDK need to be created in the top folder directory. Examples:
    c:\prj\satscan.development\satscan>mklink /D boost "c:\prj\boost\"
    c:\prj\satscan.development\satscan>mklink /D jdk "c:\Program Files\Java\jdk1.6.0_33\"
